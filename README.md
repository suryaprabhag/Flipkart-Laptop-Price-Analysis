# Flipkart Laptop Price Analysis

## Project Overview

This project performs web scraping on Flipkart laptop listings and analyzes the factors influencing laptop prices using Exploratory Data Analysis (EDA).

## Objectives

- Scrape laptop information from Flipkart
- Clean and preprocess data
- Perform EDA
- Generate business insights

## Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Regex

## Dataset

- Source: Flipkart
- Records: 500+
- Features:
  - Price
  - Brand
  - Model
  - Processor
  - RAM
  - Storage
  - Storage Type
  - Operating System
  - Screen Size
  - Customer Rating
  - Ratings Count
  - Reviews Count
  - Discount Percentage

## Key Insights

- Premium brands generally have higher prices.
- Higher RAM and Storage Capacity increase laptop prices.
- SSD is the most common storage type.
- Windows 11 is the dominant operating system.
